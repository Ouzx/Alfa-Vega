# Alfa-Vega
Revolution of INDUSTRY

Alfa-Vega klasöründe c# ile yazdığım UI bulunmakta. Veri tabanı üzerinden aldığı bilgileri doğrudan göstermekte.
Alfa-Vega-Pi klasöründe ise sensörlerin bağlı olduğu arduinoların gönderdiği veriyi parse edip veri tabanını gönderen kodlar bulunmakta. Pi ile arduinolar arasında NRF24L01 bulunmakta.
